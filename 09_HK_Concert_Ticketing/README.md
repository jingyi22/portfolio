# 香港演唱会票务平台 UX 优化方案 Demo

基于 Trip.com 演唱会门票竞品测试研究，针对票券冒领问题设计优化方案。

## 内容

- **Introduction Page**: 研究背景、4大核心问题、竞品对比矩阵
- **4个交互式 Demo Screen**: 手机模拟界面展示优化后的购票流程

## 4个核心改进点

1. **政策醒目化** — 红色警告横幅 + 独立政策卡片
2. **信息确认页** — 4项必读勾选 + 按钮在全部勾选后才可点击
3. **订单确认页** — 取票码部分隐藏 + 防截图提示
4. **订单管理页** — 票券状态展示 + 安全提示

## 技术

- 单 HTML 文件，无外部依赖
- 风格与 07_AI_Chatbot_Feedback 保持一致
- 交互式 Tab 切换 + 勾选确认逻辑
